texto 

Dos edificios que respiran al unísono, sin saberlo. Un sensor PIR en el LID, otro ultrasónico en la biblioteca — cada uno contando siluetas, midiendo presencia, traduciendo lo humano a pulsos digitales. Entre ellos, el aire no está vacío: viaja un protocolo, una petición HTTP que cruza lo inalámbrico como mensaje entre organismos vivos. Una API recibe el conteo, lo procesa, lo devuelve transformado — y un actuador responde: una pantalla que se ilumina, un LED que cambia de color, una señal que dice aquí hay vida, ven o el espacio respira solo hoy. No es solo automatización — es escucha mutua. El sistema nervioso oculto de los lugares donde pensamos, construido sobre microcontroladores, WiFi y la pregunta técnicamente precisa pero profundamente humana: ¿cuántos habitan tus pasillos ahora mismo?

¿qué sensor van a ocupar para detectar a un ser vivo?

¿cómo se comunica visualmente el conteo al usuario final? ¿qué significan los distintos colores del led?
