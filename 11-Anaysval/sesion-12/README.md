### APIs

Según Gemini: Una API (Application Programming Interface o Interfaz de Programación de Aplicaciones) es un conjunto de reglas que permite que distintos programas o servicios intercambien información entre sí.

Las APIs facilitan el desarrollo de aplicaciones porque permiten utilizar servicios ya existentes sin necesidad de programarlos desde cero. Funcionan como un puente de comunicación entre diferentes sistemas, compartiendo únicamente la información necesaria para cada solicitud.

Por ejemplo, cuando una persona realiza un pago mediante PayPal en una tienda en línea, la conexión entre la tienda y el sistema de pago se realiza a través de una API.

### Arrays

Según Gemini: Un array (o arreglo) es una estructura que permite almacenar múltiples datos del mismo tipo dentro de una sola variable. En lugar de crear muchas variables independientes, se agrupan todos los valores en un único arreglo, facilitando su organización y manipulación dentro del programa.

### Primer texto

El Oráculo, es una experiencia interactiva que invita a los participantes a formular una pregunta en voz alta. Luego, mediante un aplauso, activan una consulta hacia una entidad remota que responde con una frase, imagen o mensaje abierto a la interpretación. No busca entregar respuestas exactas, sino generar un espacio de reflexión donde cada persona construye su propio significado.
