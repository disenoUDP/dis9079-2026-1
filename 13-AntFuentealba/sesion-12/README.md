## Apuntes
### Arrays
Un array (o arreglo) es una estructura que permite almacenar múltiples datos del mismo tipo dentro de una sola variable. En lugar de crear muchas variables independientes, se agrupan todos los valores en un único arreglo, facilitando su organización y manipulación dentro del programa.

### Desarrollo del proyecto de examen
Durante la clase conversamos sobre la propuesta para el examen final y la importancia de definir qué queremos comunicar a través de nuestra obra. Más que enfocarnos únicamente en el funcionamiento técnico, trabajamos en la construcción del lenguaje conceptual del proyecto, pensando en cómo presentarlo de manera similar a un portafolio artístico.

La idea es desarrollar una experiencia interactiva que combine tecnología, comunicación inalámbrica e interpretación personal, considerando tanto los aspectos técnicos como la experiencia que tendrá el participante.

Yeseul Song es una artista nacida en Corea del Sur que trabaja con tecnología, interacción y participación como herramientas de creación artística. Su obra explora formas alternativas de percepción a través de estímulos sensoriales que van más allá de la visión.

### APIs
Una API (Application Programming Interface o Interfaz de Programación de Aplicaciones) es un conjunto de reglas que permite que distintos programas o servicios intercambien información entre sí.

Las APIs facilitan el desarrollo de aplicaciones porque permiten utilizar servicios ya existentes sin necesidad de programarlos desde cero. Funcionan como un puente de comunicación entre diferentes sistemas, compartiendo únicamente la información necesaria para cada solicitud.

Por ejemplo, cuando una persona realiza un pago mediante PayPal en una tienda en línea, la conexión entre la tienda y el sistema de pago se realiza a través de una API.

## Proyecto de examen: "El Oráculo"

Nuestro proyecto consiste en una experiencia interactiva llamada El Oráculo.

La obra invita a los participantes a formular una pregunta en voz alta. Una vez realizada la pregunta, un aplauso activa el sistema y desencadena una consulta hacia una entidad remota mediante una API.

La respuesta es devuelta en forma de frase, imagen o mensaje abierto a la interpretación. El objetivo no es entregar respuestas correctas o precisas, sino generar un espacio de reflexión personal donde cada participante construya su propio significado a partir de la respuesta obtenida.

Desde el punto de vista conceptual, la obra explora la relación entre tecnología, incertidumbre y búsqueda de sentido, utilizando la comunicación inalámbrica como un medio para establecer un diálogo entre el usuario y una inteligencia remota.
